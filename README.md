# MATLAB with Python Book

私がお話しするエンジニアや科学者は、MATLAB と Python を "MATLAB **<u>vs</u>** Python" と考えています。

本書の目的は、そんな彼らに "MATLAB **<u>with</u>** Python"として考えることが可能であることを証明することです。
